# Horror Game Development Course

Welcome to the official repository for the **Horror Game Development Course** by Gamesoft Studio.

This repository contains the source code used throughout the course. Each episode has its own folder containing only the scripts covered in that episode.

The goal of this course is to help you build a complete horror game from scratch while learning reusable game development systems in Unity.

## Repository Structure

```
Episode01/
Episode02/
Episode03/
...
```

Each folder contains only the scripts from that episode.

## Before You Download

I highly recommend watching the episode and implementing everything yourself before looking at the source code.

The purpose of this repository is to help you if:

- You get stuck while following the tutorial.
- You accidentally delete or break a script.
- You want to compare your implementation after finishing the episode.

Avoid simply copying the scripts without understanding them—you'll learn much more by building everything yourself.
